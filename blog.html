sessionStorage is similar to localStorage. The main difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends. The storage capacity of local storage is 5MB/10MB. The storage capacity of session storage is 5MB.

Global scope contains all of the things defined outside of all code blocks. A global variable has global scope. A global variable is accessible from anywhere in the code.
Block scope contains things defined inside code blocks. A block scope variable has block scope. A block scope variable is only accessible where it’s declared.

The Event Loop has one simple job — to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it.
Such an iteration is called a tick in the Event Loop. Each event is just a function callback.
